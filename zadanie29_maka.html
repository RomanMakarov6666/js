ZADANIE 29 — Pytania

  1. W jaki sposób można określić właściwość danego obiektu (dwa sposoby z przykładami)?
  2. W jaki sposób można odwołać się do pól (właściwości) i metod?
  3. Wymień cechy charakterystyczne konstruktora.
  4. Konstruowanie obiektu na podstawie przykładu — obiekt ma trzy pola oraz jedną metodę.
  5. Znaczenie słowa this. Jak oddzielamy pola oraz metody?
  6. Konstruktor w JS — przykład z opisem.
  7. Znaczenie słowa kluczowego new + przykład.
  8. Znaczenie słowa kluczowego prototype + przykład (ważny tekst pogrubiony).

ODPOWIEDZI

1. Dwa sposoby nadania właściwości

// Sposób 1: kropkowy
obj.marka = "Opel";

// Sposób 2: indeksowy
obj["cena"] = 25000;

2. Odwołanie do pól i metod

console.log(osoba.imie);
osoba.pokaz();

3. Cechy konstruktora

  • Wywoływany przy tworzeniu obiektu.
  • Nazywa się jak klasa (w JS to funkcja).
  • Nie zwraca wartości.
  • Może przyjmować argumenty.

4. Przykład obiektu z 3 polami i 1 metodą

var osoba = {
  nazwisko: "Kowalski",
  imie: "Jan",
  zawod: "piekarz",
  wyswietl: function(){ document.write(this.nazwisko + " " + this.imie); }
};

5. Znaczenie this

this wskazuje na aktualny obiekt, pozwala wewnątrz metody odnieść się do pól/metod tego obiektu.

6. Konstruktor w JS — przykład

function Klient(nazwisko, imie, zawod){
  this.nazwisko = nazwisko;
  this.imie = imie;
  this.zawod = zawod;
  this.wypisz = function(){ alert(this.nazwisko + " " + this.imie); };
}

7. Słowo kluczowe new

var k = new Klient("Kowalski","Jan","kierowca");

8. Właściwość prototype — przykład

function Klient(){ this.imie = "Paweł"; }
Klient.prototype.pisz = function(){ console.log(this.imie); };
var x = new Klient();
x.pisz();
