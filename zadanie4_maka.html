Teorii proszę szukać w tej instrukcji (kliknij).

1) kiedy powstał JavaScript (podaj datę).

JavaScript został stworzony w 1995 roku

2) dlaczego jest to język skryptowy

Ponieważ jest interpretowany w czasie działania aplikacji i nie wymaga wcześniejszej kompilacji do pliku binarnego.

3) gdzie jest wykonywany JS (klient lub serwer)

JavaScript może być wykonywany zarówno po stronie klienta (przeglądarka), jak i po stronie serwera (np. Node.js).

4) dlaczego jest JS to bezpieczne narzędzie

JavaScript działa w izolowanym środowisku (sandbox) i ma ograniczony dostęp do systemu. Dodatkowo obowiązuje polityka same-origin i inne mechanizmy bezpieczeństwa.

5) czy jest to język obiektowy, jeśli tak to dlaczego.

Tak, jest obiektowy, ponieważ opiera się na obiektach i prototypach. Od ES6 ma także składnię klas.

6) wymień zalety JS:

Zaleta1 — działa we wszystkich przeglądarkach
Zaleta2 — umożliwia tworzenie dynamicznych i interaktywnych stron
Zaleta3 — posiada ogromny ekosystem bibliotek i frameworków

7) Framework to:

Zestaw narzędzi i reguł ułatwiających tworzenie aplikacji, narzucający strukturę i porządkujący kod.

8) Do aplikacji mobilnych można użyć Framework

Tak, np. React Native, Ionic, NativeScript.

9) Aplikacje responsywne to:

Strony/aplikacje, które dopasowują wygląd do różnych urządzeń i rozdzielczości ekranu.

10) Przepisz linię kodu:

<input type="button" name="przycisk" value="Nowa Strona" onclick="WinOpen(' ')">

Wyjaśnienie: Tworzy przycisk z napisem „Nowa Strona”. Po kliknięciu wywołuje funkcję WinOpen(' ').

11) Przepisz linię kodu:

window.open("obraz.html","okienko","toolbar=no,directories=no,menubar=no,height=280,width=160,top=200,left=200");
Wyjaśnienie: Otwiera nowe okno przeglądarki o nazwie „okienko” z plikiem obraz.html i z ustawionymi opcjami.
toolbar=no — ukrywa pasek narzędzi
directories=no — ukrywa katalogi (obecnie ignorowane)
menubar=no — ukrywa menu
height=280 — wysokość okna 280px
width=160 — szerokość okna 160px
top=200 — odległość 200px od góry ekranu
left=200 — odległość 200px od lewej krawędzi ekranu

12) Opisz następującą instrukcję window.close()

Wyjaśnienie: Zamyka bieżące okno przeglądarki. Zadziała tylko, jeśli okno zostało otwarte przez skrypt.
